# Push maison A
| Exercice                               | Séries x reps      | Repos   | Rôle                                                    |
| -------------------------------------- | ------------------ | ------- | ------------------------------------------------------- |
| Pompes déclinées                       | 4 x 8-20           | 2-3 min | Pecs hauts, triceps, bon mouvement de base à la maison. |
| Développé haltères au sol              | 3 x 8-15           | 3 min   | Remplace le chest press, pecs + triceps.                |
| Pike push up                           | 3 x 6-15           | 2-3 min | Épaules, surtout deltoïdes antérieurs.                  |
| Extension triceps au-dessus de la tête | 3 x 10-20          | 2-3 min | Triceps, surtout longue portion.                        |
| Élévations latérales haltères          | 3 x 12-20          | 2 min   | Deltoïde moyen.                                         |
| Pompes serrées                         | 2 x échec contrôlé | 2 min   | Finisher triceps.                                       |

| Low row                      | 3 x 6-8       | 3 min   | Tirage horizontal pour le milieu du dos.                    |

| Hammer curl                  | 3 x 8-12      | 2-3 min | Brachial, brachio-radial, avant-bras, biceps.               |
