# Workout Session Showcase

A small personal project that lets me quickly view the workout sessions I can do depending on the context: gym or home, push, pull, and legs.

## Project contents

- `index.html`: site structure.
- `style.css`: simple and modern black-and-white styling.
- `script.js`: display logic for the sessions.
- `.md` files: session summaries, useful if anyone wants to reuse or adapt the template.

## Goal

This site acts as a showcase for my available workout sessions.
It is not meant to be a full application, just a clean and practical way to browse the routines.
